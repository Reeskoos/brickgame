# BrickGame

BrickGame v2.0 is a set of classic games that includes the snake game and Tetris. 
The program is developed in C++ using the C++17 standard.

## Requirements

- OS: Linux or MacOS
- C++17 compiler
- Cmake
- Qt5 and higher.
- Ncurses, gtest libs

## Usage
 1. Run "make install".
 3. Run brick_game_desktop or brick_game_console.
